## 一、各Agent数据需求与缺失指标

### 1. Technical Analyst（技术分析师）

| 项目 | 内容 |
|------|------|
| 投资风格 | 多维技术交易策略（趋势跟踪、均值回归、动量、波动率、统计套利） |
| 核心数据 | OHLCV日线价格数据 |
| 派生指标 | EMA、ADX、RSI、Bollinger Bands、ATR、Hurst指数 |
| 缺失指标 | 无（完全自计算） |
| 影响程度 | 🟢 低 |

### 2. Fundamentals Analyst（基本面分析师）

| 项目 | 内容 |
|------|------|
| 投资风格 | 盈利能力 + 增长 + 财务健康 + 估值比率综合分析 |
| 核心数据 | FinancialMetrics: ROE、净利率、营业利润率、收入增长、EPS增长、流动比率、D/E、P/E、P/B |
| 缺失指标 | 无明显缺失 |
| 影响程度 | 🟢 低 |

### 3. Sentiment Analyst（情绪分析师）

| 项目 | 内容 |
|------|------|
| 投资风格 | 内部人交易信号 + 新闻情绪加权 |
| 核心数据 | 内部人交易记录、公司新闻 |
| 缺失指标 | 内部人持仓比例变化率、新闻情绪置信度细节 |
| 影响程度 | 🟡 中 |

### 4. News Sentiment（新闻情绪Agent）

| 项目 | 内容 |
|------|------|
| 投资风格 | LLM驱动的新闻标题情绪分类 + 置信度聚合 |
| 核心数据 | 公司新闻标题、来源、日期 |
| 缺失指标 | **新闻完整正文**、新闻热度/阅读量、行业对标新闻流 |
| 影响程度 | 🟡 中 |

### 5. Valuation Analyst（估值分析师）⚠️

| 项目 | 内容 |
|------|------|
| 投资风格 | 四方法估值聚合（DCF + Owner Earnings + EV/EBITDA + 残余收入） |
| 核心数据 | 10期TTM财务数据、市值、FCF历史 |
| 缺失指标 | **Beta值**（当前hardcode=1.0）、**无风险利率**（hardcode=4.5%）、**市场风险溢价**（hardcode=6%） |
| 影响程度 | 🔴 高 |

### 6. Growth Analyst（增长分析师）

| 项目 | 内容 |
|------|------|
| 投资风格 | 历史增长趋势 + 增长估值 + 边际扩张 + 内部人信号 |
| 核心数据 | 12期TTM数据（3年趋势）、内部人交易 |
| 缺失指标 | 毛利率趋势数据、行业平均增长对标 |
| 影响程度 | 🟡 中 |

### 7. Warren Buffett（沃伦·巴菲特）⚠️

|------|------|
| 投资风格 | 经济护城河 + 管理质量 + 内在价值 + 安全边际 |

| 缺失指标 | **ROIC**（护城河评估核心）、商誉摊销率、持续价值增长率 |

## 六、总结与建议

### 当前状态
- **完全可用**: 5个Agent（Technical、Fundamentals、Ben Graham、Phil Fisher、Bill Ackman）
- **部分受限**: 9个Agent（缺少增强指标但基础功能可用）
- **严重受限**: 5个Agent（Valuation、Damodaran、Nassim Taleb、Warren Buffett、Cathie Wood）

### 推荐路径
1. **立即实施方案A**（3人天）：解决Beta、无风险利率、ROIC三大核心缺口，让Valuation和Damodaran两个关键Agent的估值从"猜测"变为"计算"
2. **评估后实施方案B**（额外17人天）：全面提升13个Agent的分析质量
3. **按需评估方案C**（额外33人天）：根据实际使用反馈决定是否投入

### 投入产出比
- 方案A以**极小投入**解决**最大痛点**，ROI最高，建议优先实施
- 方案B的行业对标和PEG功能对多位大师Agent有显著提升
- 方案C中TAM和竞争格局属于定性数据，自动化成本高，建议谨慎评估
